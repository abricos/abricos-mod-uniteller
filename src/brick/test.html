Система Uniteller

Изменение статуса заказа

Все поля в этой форме заполняются автоматически. Для этого необходимо открыть эту страницу по формату: http://mydomain.tld/payments/test/uniteller/[ORDERID]/[STATUS]/.
Например: http://abricos.local/payments/test/uniteller/fddfc109a420516bb93bf22c6104343c/paid/

POST-форма для проверки запроса платежной системы на изменение статуса заказа.

В настройках магазина системы Uniteller в поле "URL для уведомление сервера интернет-магазина об изменившемся статусе счёта/оплаты" необходимо указать значение по формату: http://mydomain.tld/payments/api/uniteller/
